What is NEQR?


NEQR (Novel Enhanced Quantum Representation) is a quantum image model that encodes grayscale pixel values into quantum states. When combined with AES encryption, it provides a strong hybrid security mechanism that allows both quantum representation and classical encryption for secure image transmission.
